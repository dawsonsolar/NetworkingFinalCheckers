# Networking Final Project — Networked Checkers

Final project for CSI-275 / GPR-430 Networking at Champlain College.

A turn-based multiplayer checkers game built in C++ with a browser-based client. Demonstrates authoritative server architecture, custom TCP protocol design, session persistence, and WebSocket bridging.

**Play it here:** https://solar722.itch.io/networking-final-checkers

---

## Project Structure

```
checkers-net/       C++ game server + Node.js web bridge + browser client
```

See [`checkers-net/README.md`](checkers-net/README.md) for build and run instructions.
